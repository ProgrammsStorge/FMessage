# FMessage

# О чем проект?
Этот проект позволяет обмениваться текстом и файлами внутри локальной сети. Проект нацелен на учебные заведения, в которых есть потребность в обмене сообщениями между компьютерами. 
Упор в проекте делается на безопасность и скорость передачи данных.
