# О чем проект?
Этот проект позволяет обмениваться текстом и файлами внутри локальной сети. Проект нацелен на учебные заведения и компаний, в которых есть потребность в обмене сообщениями между компьютерами. 
Упор в проекте делается на безопасность и скорость передачи данных.

## Почему вам стоит выбрать нас?
* **Удобство**: Вам не нужен сервер для мессенджера!
* **Безопасность**: Ваши сообщения никуда и некому не отправляются!
* **Умная система уведомлений**: Пользователи могут не отвлекаться на ненужные сообщения.
* **Дизайн**: Дизайн не перегружен и интуитивно понятен.
* **Открытки на каждый день**: Наш мессенджер может полностью заменить социальную сеть "одноклассники".

## Безопасность
* **Шифрование**: Шифрование сообщений и файлов.
* **Режим белого списка**: Пользователь не сможет связаться с тем кого не знает.
* **Проверка на фишинг**: Пользователь защищён от вредоносных ссылок.

## Умная система уведомлений
Мессенджер - Завлекают. Пользователь заходит проверить сообщения, и теряет фокус на задаче. Переключение между мессенджером и работой тратит энергию, что ведёт к падению продуктивности. Наша система избавляет от привычки постоянно переключатся между работой и мессенджером. Система сама отфильтрует приоритетные оповещения, помогая сохранять концентрацию и работать эффективнее.

## Стек проекта
* **Ядро**: Python 3.11
* **База данных**: SQLite3
* **Интерфейс**: Flet
* **Серверная часть**: Flask
* **Безопасность**: Crypto
* **Прочее**: requests

## Экономическое влияние
Экономика будет основываться на B2B, этот проект предназначен для компаний, но может использоваться в других целях.
