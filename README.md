# О чем проект?
Этот проект позволяет обмениваться текстом и файлами внутри локальной сети. Проект нацелен на учебные заведения, в которых есть потребность в обмене сообщениями между компьютерами. 
Мессенджер работает по системе Peer-to-Peer, что означает, что компьютеры выступают в роли клиента и сервера одновременно. Peer-to-Peer система позволяет мессенджеру работать без центрального сервера. 
![alt text](https://github.com/ProgrammsStorge/FMessage/blob/main/images/p2pshow.png?raw=true)
Упор в проекте делается на безопасность и скорость передачи данных. В проекте реализовано RSA-шифрование.
